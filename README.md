Live Dashboard Link -: [https://app.powerbi.com/groups/me/reports/c2ccbbbd-e714-419d-88bc-dfd9dc55490c/0db75b6eb80646b44e71?experience=power-bi]


Project Overview
The goal of this project was to create a single source of truth for AtliQ Hardware's stakeholders. It integrates data from various departments to provide actionable insights.

Tech Stack & Skills
Tool: Power BI Desktop

Data Source: MySQL / Excel

DAX: Created 50+ complex measures (YTD, PY, Margin %, etc.)

Data Modeling: Implemented a Star Schema with 10+ tables.
